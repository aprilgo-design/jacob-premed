# Jacob's Path to Medicine 🩺

This repository contains a comprehensive, personalized planning guide for **Jacob’s journey from Grade 10 to Medical School in Canada**. 

## 📋 Overview
The project centralizes academic roadmaps, financial breakdowns, and scholarship tracking specifically tailored for Ontario pre-med programs.

### Key Features:
* **Program Comparisons:** Side-by-side analysis of McMaster BHSc, Queen's Health Sciences, Western BMSc, and UofT Life Sciences.
* **Math vs. Science Paths:** Evaluating traditional science routes against math-heavy alternatives (iBioMed, Bioinformatics).
* **Financial Planner:** Real-time cost estimates per semester, including tuition, residence, and meal plans.
* **Funding Tracker:** Projections for RESP growth, OSAP loan estimates, and a curated list of automatic and application-based scholarships.
* **ADHD Support:** Dedicated section on accessibility services and disability-specific grants.
